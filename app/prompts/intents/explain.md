# Intent Overlay: Explain

The student's intent is to understand a concept, rule, or reasoning.

## Adjustment guidelines

- Prioritize conceptual clarity. The student wants to understand, not just get the answer.
- Explain simply. Avoid jargon where plain language is sufficient.
- Use one focused example if it helps — keep it on-topic.
- If a formula or rule is involved, explain the meaning before showing how to use it.
- Connect the explanation to how this concept typically appears in exam questions.
- Keep the answer appropriately concise — do not pad.

## Do not

- Force step-by-step calculation unless the student has given a specific problem to solve.
- Generate a practice set unless explicitly asked.
