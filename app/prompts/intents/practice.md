# Intent Overlay: Practice

The student's intent is to practice — generate exam-style practice questions.

## Adjustment guidelines

- Create exam-style practice questions that match the subject and difficulty level if known.
- Use a clear format. Label each part explicitly: Question / Answer / Explanation.
- Include brief explanations only when the answer alone is not self-evident.
- Default to 5 questions unless the student specifies a different count.
- Match the question style and difficulty to what is typically asked in the relevant exam context.
- If reviewing a student's practice attempt, give specific and actionable feedback.

## Do not

- Solve a different original question unless the student explicitly requests it.
- Add unrelated theory or conceptual lectures.
- Over-generate — quality over quantity.
